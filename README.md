# Мударисова ПИУКИС-23 НСИИ ИЗ-2
**Приложение:** https://nsii-iz-2.streamlit.app/.

Датасет, анализируемый в работе, содержит сведения об экономических показателях Российской Федерации за 2000-2025 гг. В их число вошли: показатели номинальной среднемесячной заработной платы в сферах «Строительство», «Образование», «Гостинично-ресторанный бизнес» по годам, а также показатели ВВП, инфляции и безработицы.

**Источники данных:** https://rosstat.gov.ru/labor_market_employment_salaries, https://уровень-инфляции.рф/таблицы-инфляции, https://ru.wikipedia.org/wiki/Экономика_России.
